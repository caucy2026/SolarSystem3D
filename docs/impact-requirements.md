# Jupiter Impact Simulator — 需求规格文档

> 基于用户交流记录整理，用于指导撞击特效 Demo 和 APP 集成开发。

---

## 1. 核心原则

- **物理级真实感**：效果应接近科学可视化，不是玩具特效
- 参考真实撞击模拟器模型，非凭空想象
- 拒绝卡通风格：白色亮环 ❌、灰色椭圆饼 ❌、彩色塑料环 ❌
- 效果精而不少：每个效果都要经得起细看

---

## 2. 参考开源项目

| 项目 | 用途 | 说明 |
|------|------|------|
| **Earth Impact Effects Program** (Imperial College) | 撞击参数 | 陨石坑大小、火球持续时间、喷射物速度 |
| **iSALE** | 冲击物理模型 | 陨石坑剖面形状、冲击波传播 |
| **EasyCraterSim** | 陨石坑剖面 | `d<0.25: deep bowl → transition → raised rim` |
| **WebGL Explosion VFX** | 火球着色器 | 噪声位移 + 热辐射渐变 |
| **FireExplosionShader** | 热辐射 | 白核→橙→暗红色彩映射 |
| **Gravity-assist-simulation** | 引力捕获 | 小行星被木星引力捕获的物理逻辑 |

## 3. 效果规格

### 3.1 撞击火球 (Fireball)

**触发时机**：小行星接触木星表面瞬间

**视觉要求**：
- 体素感——噪声位移顶点，非光滑球体
- 色彩：核心白热→中层橙→外层暗红
- 膨胀：先快后慢，约 5s 生命周期
- 半透明 AdditiveBlending

**Shader 技术**：
```
顶点：5层 FBM 噪声位移，每层频率加倍、振幅减半
片段：径向距离驱动颜色混合
  d=0.0~0.3 → 白热 core (1.0,0.95,0.85)
  d=0.3~0.7 → 橙 mid (1.0,0.4,0.05)
  d=0.7~1.0 → 暗红 outer (0.6,0.05,0.01)
  叠加噪声微调色相
```

### 3.2 撞击坑 (Impact Crater)

**触发时机**：火球出现后 1.5s 开始显现

**视觉要求**：
- 基于 EasyCraterSim 物理剖面
- 深碗形中心凹陷
- 过渡区（碗→平地）
- 轻微隆起边缘
- 暗色材质（烧焦外观）
- 持续可见 3+ 分钟

**几何技术**：
```
圆盘投影到球面（geodesic patch）
顶点高度 = crater_profile(distance_from_center)
  if d < 0.25: h = -depth * (1 - d/0.25)         // deep bowl
  if 0.25 < d < 0.55: h = transition              // smooth transition
  if 0.55 < d < 1.0: h = rim_height * (d-0.55)/0.45  // raised rim

深度 ≈ 0.18 × 直径
边缘高度 ≈ 0.03 × 直径
```

### 3.3 引力波 (Gravitational Waves)

**触发时机**：撞击瞬间，与火球同时

**视觉要求**：
- 从撞击点向**外太空**扩散（不是沿木星表面）
- 薄环状波前——只在扩展前沿可见，非填充球体
- 干涉条纹模拟空间涟漪
- 蓝白冷色调，模拟引力透镜效果
- Fresnel 效果：边缘比中心更亮
- 4 层交错波，每层延迟 ~0.5s
- 扩散 6-8s 后渐隐消失

**Shader 技术**：
```
球壳 mesh（完整球面，FrontSide）
fragment shader:
  ring = exp(-(dist - waveRadius)² * 12)  // 薄环 at wavefront
  ripple = sin(dist*25 - time*3.5)*0.4 + sin(dist*15 + time*2)*0.3
  fresnel = pow(1 - dot(normal, viewDir), 2.0)
  alpha = ring * (0.5 + ripple*0.3) * fresnel * uAlpha * 0.55
  color = (0.35, 0.55, 0.9) + ripple * (0.3, 0.25, 0.15)
```

### 3.4 喷射物 (Ejecta)

**触发时机**：撞击瞬间

**视觉要求**：
- 100-150 个粒子
- 各向异性速度分布（锥形喷射）
- 受木星引力抛物线轨迹
- 颜色从亮白热→橙→暗红冷却变化
- 3-8s 生命周期

**物理参数**：
```
速度：0.3~2.5 × power（场景单位/秒）
仰角：0~45°
木星引力加速：gf = 2.5*dt / (dist²*0.3 + 1)
拖拽：0.998/帧
```

### 3.5 动态光照 (Flash Lighting)

**触发时机**：撞击瞬间

**视觉要求**：
- PointLight 瞬间亮起（暖橙色）
- 快速衰减（~2s）
- 照亮火球和周边碎片

## 4. 效果时间线

```
t=0.0s  ─┬─ 火球诞生（0.25 单位） + 引力波第1层
         ├─ 喷射物发射（150粒子）
         ├─ 点光源闪烁（强度 60×power）
         │
t=0.5s  ─┬─ 引力波第2层
         │
t=1.0s  ─┬─ 引力波第3层
         │
t=1.5s  ─┬─ 撞击坑开始显现（渐显至目标不透明度）
         ├─ 引力波第4层
         │
t=3.0s  ─┬─ 火球开始衰减
         │
t=5.0s  ─┬─ 火球消失
         │
t=7.0s  ─┬─ 引力波全部消失
         │
t=180s  ─┴─ 撞击坑最终淡出
```

## 5. 已知拒绝方案

| 方案 | 拒绝原因 |
|------|----------|
| 白色 torus 冲击波环 | "太违和"——不符合物理规律 |
| 半切半球引力波 | "椭圆灰饼"——视觉丑陋 |
| 彩色塑料环 | "小孩子写的DEMO"——不够严肃 |
| 简单球体缩放 | 缺乏体积感 |
| 蓝色玩具环（旧版 grav wave） | "太卡通" |
